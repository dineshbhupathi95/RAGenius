# RAGenius
RAGenius is an intelligent document question-answering system built using the Retrieval-Augmented Generation (RAG) architecture. The application allows users to upload documents such as PDFs or text files and ask questions in natural language.

The system processes documents by splitting them into chunks, generating embeddings, and storing them in a vector database. When a user submits a query, the system retrieves the most relevant document chunks and sends them to a Large Language Model (LLM) to generate context-aware answers.

This project demonstrates how to build scalable AI applications by integrating modern technologies like vector search, LLMs, and retrieval pipelines.
